#Tank
TANK Game - PyGame Project
Описание

Простая 2D игра на PyGame, где игрок управляет танком и сражается с врагами.
Управление

    A - движение влево

    D - движение вправо

    SPACE - стрельба

    ESC - выход из игры

Требования

    Python 3.x

    PyGame библиотека

Установка

    Установите Python 3.x

    Установите PyGame:
    text

    pip install pygame

    Запустите игру:
    text

    python tank_game.py

Структура игры
text

tank_game.py - основной файл игры

Геймплей

    Игрок управляет танком в нижней части экрана

    Враги появляются в верхней части экрана и движутся вниз

    Цель - уничтожать врагов стрельбой

    При столкновении с врагом игра заканчивается

Разработка

Игра создана в образовательных целях для изучения PyGame и программирования игр на Python.
